В поле `ORDER_CHANNEL_ID` вам нужно указать идентификатор Telegram-канала, который вы будете использовать для отправки заказов.

Вот как его получить:

**1. Создайте Telegram-канал**

Если у вас его еще нет, создайте новый канал в Telegram. Он может быть как публичным, так и частным.

**2. Добавьте ботов в канал**

Добавьте обоих ботов (клиентского и водительского) в этот канал в качестве администраторов. Это необходимо, чтобы один бот мог отправлять сообщения в канал, а другой — читать их.

**3. Получите ID канала**

*   **Для публичного канала:** Если ваш канал публичный, его ID — это его юзернейм (например, `@my_channel_name`).

*   **Для частного канала:** Если ваш канал частный, ID будет числовым. Чтобы его узнать, вы можете:
    1.  Переслать любое сообщение из вашего канала боту `@userinfobot`.
    2.  Бот в ответ пришлет информацию о канале, включая его ID (это будет отрицательное число, начинающееся с `-100...`).

**Пример:**

Если у вас публичный канал с юзернеймом `toptaxichannel`, то в `config.json` вы напишете:
```json
"ORDER_CHANNEL_ID": "@toptaxichannel"
```

Если у вас частный канал и `@userinfobot` показал ID `-1001234567890`, то вы напишете:
```json
"ORDER_CHANNEL_ID": "-1001234567890"
```
**Важно:** Для частных каналов ID должен быть в виде строки, а не числа.
